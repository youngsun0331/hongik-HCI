웹 폼 만들기

웹 폼은 사용자로부터 데이터를 입력받아 서버로 전달하는 핵심적인 인터페이스입니다.

form 태그의 핵심 속성

name
폼의 이름을 지정하며, 자바스크립트나 서버에서 폼을 식별할 때 사용합니다
action
폼 데이터를 처리할 서버 측 응용 프로그램의 URL 주소를 명시합니다.
method
데이터 전송 방식을 결정하며, 주로 GET(주소창 노출) 또는 POST(본문 포함) 방식을 사용합니다.

주요 입력 요소의 기능

  • text & password: 한 줄 짜리 일반 텍스트 또는 암호화된 문자를 입력받는 필드입니다.
  • checkbox: 여러 항목 중에서 한 개 이상의 값을 중복 선택할 때 활용합니다.
  • radio: 나열된 여러 항목 중 단 하나만을 선택해야 할 때 사용하며, 같은 'name' 속성을 공유해야 합니다.
  • textarea: 자기소개서나 의견처럼 여러 줄의 긴 문장을 입력받기에 적합합니다.

폼의 구조화

폼 요소를 논리적으로 그룹화할 때는 <fieldset> 태그를 사용하며, 그 그룹의 제목은 <legend> 태그로 정의하여 사용자 편의성을 높입니다.
