배치

CSS3를 이용해 HTML 태그가 브라우저 화면에 출력되는 위치를 결정합니다.

위치 지정
  1. static: 태그가 작성된 순서대로 배치되는 기본 방식입니다.
  2. relative: 본래 있어야 할 위치를 기준으로 상하좌우 이동합니다.
  3. absolute: 부모 요소를 기준으로 절대적인 좌표에 위치시킵니다.
  4. fixed: 스크롤을 내려도 화면의 특정 위치에 항상 고정됩니다.
배치 관련 추가 속성
  • float: 요소를 페이지 왼쪽이나 오른쪽으로 밀어 배치합니다.
  • z-index: 요소가 겹칠 때 어떤 것이 앞에 보일지 층을 정합니다.
  • overflow: 박스보다 큰 내용물이 있을 때 숨기거나 스크롤을 만듭니다.

리스트 꾸미기

목록 태그(ul, ol)의 스타일을 취향에 맞게 변경할 수 있습니다.

  • list-style-type: 불렛의 모양을 원, 사각형, 숫자 등으로 정합니다.
  • list-style-image: 불렛 대신 사용자 지정 이미지를 불러와 사용합니다.
  • list-style-position: 목록의 불렛을 박스 안쪽 혹은 바깥쪽에 둡니다.
