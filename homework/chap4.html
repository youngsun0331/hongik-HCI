CSS3 스타일 시트 개요

CSS란?
HTML 문서의 색상이나 모양 등 외관을 꾸미는 도구입니다.
장점
콘텐츠(HTML)와 디자인(CSS)을 분리하여 유지보수가 쉽습니다.

CSS3 스타일 시트 만들기

스타일 시트를 적용하는 세 가지 방법이 있습니다.

  1. 외부 스타일 시트 (External) - .css 파일 연결
  2. 내부 스타일 시트 (Internal) - <style> 태그 사용
  3. 인라인 스타일 (Inline) - 태그 내 style 속성 사용

셀렉터, 색과 텍스트 꾸미기

원하는 요소만 선택하여 꾸미기 위해 다양한 셀렉터를 사용합니다.

  • 전체 셀렉터 및 태그 셀렉터
  • 클래스(.) 및 ID(#) 셀렉터
  • 자식 및 자손 셀렉터

박스 모델 시각적 효과

모든 HTML 태그는 사각형의 박스 모델을 가집니다.

  • Margin: 박스와 박스 사이의 외부 간격
  • Padding: 테두리와 내용 사이의 내부 간격
  • Visual Effects: 그림자(box-shadow)와 둥근 테두리(border-radius) 효과
